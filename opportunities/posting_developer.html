Employment Opportunity: Software Engineer/Tester

Salary Range $10.35 - $12.00 per hour; DOE
Close Date 29-September-2015
Job Description We require a programmer with experience in object oriented programming (preferably Ruby or Java) and with an interest in quality assurance. This project requires regular interaction with users, short release cycles, and close collaboration with other computational HPS team members. The programmer will be responsible for extending existing software packages, for example by improving connectivity with other tools, and perform quality assurance tasks including defining and writing test cases, perform manual testing as well as setting up automatic tests. Introduction to the project and guidance will be provided; the ideal candidate will be self-motivated, autodidactic, and able to complete tasks independently.
Essential Duties
  • Write and document source code in Java, Ruby, and/or Python.
  • Manual testing of software. Writing unit and integration tests.
  • Limited server administration tasks.
  • Assist with code reviews.
  • Participate in bi-weekly (every two weeks) iteration meetings.
Desired Qualifications

Required

  • Completed at least two years of undergraduate CS training.
  • Competent programming in Ruby and/or Java.
  • Experience with software design.
  • Excellent communication and teamwork skills.
  • Experience with the software development IDEs.
  • Experience with version control systems such as GitHub.

Preferred

  • Graduate student in Computer Science, or equivalent training is preferred.
  • Experience with automatic testing frameworks such as Cucumber.
  • Experience with Jenkins CI.
